Chat with you video using Gemini AI


## Prerequisites

- Python 3.7 or higher
- Git

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/shershah1024/Gemini_video_chat.git
   ```

2. Navigate to the project directory:
   ```
   cd Gemini_video_chat
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Configuration

1. Copy the `.env.example` file to `.env`:
   ```
   cp .env.example .env
   ```

2. Open the `.env` file and replace `your api key` with your actual Gemini API key:
   ```
   GEMINI_API_KEY=your_actual_api_key_here
   ```

run python main.py
